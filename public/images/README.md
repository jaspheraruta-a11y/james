# Images Directory

## GCash QR Code

Please place your GCash QR code image in this directory with the filename: `gcash-qr-code.png`

The image will be displayed in notifications when a permit is approved and payment is completed.

### Supported formats:
- PNG (recommended)
- JPG/JPEG
- SVG

### Recommended size:
- 200x200 pixels or larger
- Square aspect ratio works best

### How to update:
1. Replace the file `gcash-qr-code.png` in this directory
2. The notification system will automatically use the new image
